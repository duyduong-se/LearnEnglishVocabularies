## Từ học hôm nay

### 1. delete *(verb — regular)*
**Nghĩa:** xóa

**Chia thì:**
| V1 | V2 | V3 |
|---|---|---|
| delete | deleted | deleted |

**Lưu ý:**
- Regular — thêm -d (vì tận cùng bằng -e)

**Cấu trúc & Ví dụ:**

delete + noun → xóa cái gì
- Please delete all unnecessary files to free up storage space.
- He deleted the old records to make room for new ones.

delete permanently → xóa vĩnh viễn
- She accidentally deleted the entire project folder permanently.
- The admin permanently deleted inactive user accounts.

delete a file/account → xóa tệp/tài khoản
- The system will delete inactive accounts after 90 days.
- You can delete your account at any time from the settings menu.

**Collocation phổ biến:**
- delete a file, delete an account, delete permanently

regularly back up → sao lưu thường xuyên
- The system will automatically back up all data every night.

**Collocation phổ biến:**
- back up data, backup files, create a backup
